Stop re-explaining yourself.

Spark loads your profile, memories, and recent sessions on every launch. Every conversation starts where the last one ended—no pasting, no catching up.

  • ✓
    Learns as you talk

    Mention your stack, your goals, or your preferences once. They're tagged as [MEMORY] and saved for next time.

  • ✓
    Picks up where you left off

    Spark summarizes recent conversations and carries them forward. Come back days later—it still has context.

  • ✓
    You own your data

    Everything is plain .md files—read, edit, or delete them anytime in any editor.

  • ✓
    Clean context, no noise

    Duplicates are caught before they're saved—your context stays clean without manual pruning.

context assembly 
// On every launch, Spark builds context from:

workspace/
├── context/
│   └── system_prompt.md      # base instructions
├── memories/
│   ├── USER.md               # who you are
│   ├── MEMORY.md             # learned facts
│   └── PREFERENCES.md        # how you work
└── sessions/
    ├── session_0218_001.jsonl
    └── session_0219_001.jsonl

// ↓ assembled into one system prompt
// ↓ sent before your first message

system_prompt = [
  base_instructions,
  user_profile,
  learned_memories,
  preferences,
  recent_sessions  // last 3
].join("---")

Everything happens in the terminal.

No electron apps. No browser tabs. A fast, lightweight CLI with persistent memory and an extensible tools ecosystem.

Persistent Memory

Mention something once and it's saved. Your facts carry forward across every session—no re-explaining.

Session History

Past conversations are summarized and loaded into every new chat. Come back days later and it still has context.

Slash Commands

Type /memories or /profile to inspect your context. Fuzzy-filtered palette with keyboard nav.

Fast Startup

Your profile, memories, and sessions are assembled into one prompt on launch. No database, no daemon—just files.

Plain Markdown

Memories, preferences, and profile are plain .md files. Open them in any editor. No proprietary formats.

Extensible by Design

Add tools, skills, and custom slash commands. TypeScript + Ink TUI—fork it, extend it, make it yours.

Up and running in 60 seconds.

All you need is Bun and an Anthropic API key.

1 Clone & install

$ git clone https://github.com/jcottam/spark.git

$ cd spark && bun install

2 Set your API key

$ mkdir -p ~/.config/spark

$ echo "ANTHROPIC_API_KEY=sk-ant-..." \

  > ~/.config/spark/.env

3 Register globally

$ bun link

→ Registered spark in ~/.bun/bin

4 Start chatting

$ spark

✓ Context assembled. Ready to chat.
